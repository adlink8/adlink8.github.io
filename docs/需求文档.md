## 详细需求规格

### 1. 主页活跃度折线图（紫色）

**目标**: 在主页右侧显示紫色折线图，实时显示"天数"活跃度

**图表规格**:
- X轴: 天数 (Days)，范围 0-31
- Y轴: 贡献度 (Contributions)
- 颜色: 折线和点为紫色，背景为深色
- 数据: 展示 "adlink" 的贡献度随时间变化趋势

### 2. 日历热力图（紫色系）

**目标**: 在折线图下方显示年度创作指数时间分布

**图表规格**:
- X轴: 月份 (10月、11月、12月、1月、2月、3月...)
- Y轴: 星期 (每行一周，从上到下)
- 单元: 每个小方块代表一天
- 颜色映射: 浅紫/灰白(不活跃) → 深紫(活跃)

**示例数据分布**:
- 10月: 月中一个浅紫色方块
- 11月: 全空白
- 12月: 月中和月末零星浅紫色方块
- 1月: 上旬一个深紫色方块(峰值)，中下旬深紫+浅紫方块
- 2月: 上旬和中旬零星浅紫色方块
- 3月: 月底一个浅紫色方块

### 3. 数据总览卡片

**目标**: 展示年度创作成果

**布局结构**:
- 整体: 圆角矩形卡片
- 背景: 深蓝色

**标题区**:
- 主标题: "数据总览" (大号白色字体)
- 副标题: "你的每一份成长都被记录" (小号浅灰色)
- 时间标签: "本年"

**数据指标区** (四个半透明浅蓝圆角矩形):
| 指标 | 说明 |
|------|------|
| 创作天数 | 数字待填充 |
| 创作字数 | 数字待填充 |
| 内容更新 | 数字待填充 |
| 获得点赞 | 数字待填充 |

### 4. 月数活跃度折线图

**目标**: 显示"月数"活跃度趋势

**图表规格**:
- X轴: 天数 (Days)，范围 0-31
- Y轴: 贡献度 (Contributions)
- 颜色: 紫色折线和点，深色背景
