# DevOps 学习笔记

这是一个记录 DevOps 和基础设施即代码学习旅程的技术博客。

## 简介

本博客基于 [Hugo](https://gohugo.io/) 静态站点生成器构建，使用 [PaperMod](https://github.com/adityatelange/hugo-PaperMod) 主题。

**主题配色：** 紫色与黑色

## 内容分类

| 分类 | 说明 |
|------|------|
| daily | 时间线/每日进度 |
| pitfalls | 实验笔记/问题排查 |
| insights | 架构深入分析 |
| reflections | 回顾与反思 |
| project-logs | 项目构建日志 |

## 部署

通过 GitHub Actions 自动部署至 GitHub Pages。
